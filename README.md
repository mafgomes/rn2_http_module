# rn2-http-module

Parte relativa a requisições HTTP(S) do projeto final de React Native 2 IESB (2023)

## Installation

```sh
npm install rn2-http-module
```

## Usage

```js
import { multiply } from 'rn2-http-module';

// ...

const result = await multiply(3, 7);
```
